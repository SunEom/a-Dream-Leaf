## 프로젝트 개요

 매년 GDP 상승과 더불어 국민 총소득도 증가하고 있는 반면 우리 사회 곳곳에서 이에 해당되지 못한 이들이 너무나도 많은 것이 현실이다. 경제적인 어려움을 겪는 많은 사람들이 있지만 그 중에서도 건강하고 다양한 영양을 섭취하여야하는 성장기의 아이들 중 한끼 식사도 제대로 챙겨먹지 못하는 결식아동들이 많다는 것을 알게 되었다. 이런 안타까운 사회 상황 속에서 지속가능하며 더불어 사는 사회를 만들어 가는 것에 동참하기위해 결식아동들을 위한 과제를 수행하고자 한다. 

 결식 아동들을 위해 다양한 정보를 제공하고 사회적 인식 변화를 위한 적극적인 지원방안을 홍보를 진행하는 것을 우선적인 목표로 삼는다. 생각보다도 많은 가게에서 결식 아동들을 위해 무상으로 다양한 서비스를 제공하는 가게들이 많다. 하지만 아이들은 이런 서비스을 알기도 힘들고 이를 직접 가게에 물어보는 것은 더욱 더 어려운 일일 것이다. 따라서 아이들이 이와 관련한 정보들을 쉽게 얻을 수 있고, 서로 공유할 수 있는 공간을 만들어준다면 아이들에게 조금은 도움이 될 수 있을 것이라고 생각한다. 

## 기술 스택

### iOS
* Swift, UIKit
* RxSwift
* MVVM
* Alamofire
* SnapKit
* Firebase Auth / Cloud Messaging
* Kakao Address, Naver Map

### Backend
* Java
* Spring Boot
* MySQL
* AWS EC2
